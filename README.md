### Run Locally

To run **News Cafe** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Xanderyeng/news-cafe.git
```

Windows:

```bash
git clone https://github.com/Xanderyeng/news-cafe.git
```
